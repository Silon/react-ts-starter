## :sparkles: Icons

To add a new icon, put svg file into `src/assets/icons` and use the naming convention `icon-[ICON_NAME]`. Remember to
make sure all stroke and/or fills are set to **currentColor** unless you are sure that's not what
you want. Finally, run this command and your icon will be available.

```shell
  yarn icons
```

## :gear: CLI Components Generating

Components may be automatically generated by plop and placed inside src/components dir:

```shell
  yarn generate
```
